# 🏏 CricStars

A full-stack cricket statistics and live score web application that provides real-time match updates, player statistics, team rankings, and graphical performance analysis.

## 🚀 Features

- 🏏 Live Cricket Scores
- 📊 Player Statistics
- 🌍 ICC Rankings
- 👥 Team Statistics
- 📅 Match Schedule
- 📈 Graphical Performance Analysis
- 🔍 Search Players and Teams

## 🛠 Tech Stack

### Frontend
- React.js
- HTML5
- CSS3
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MongoDB

### APIs
- CricAPI

## Installation

### Clone Repository

```bash
git clone https://github.com/mayankvyas-13/CricStars.git
```

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

```bash
cd frontend
npm install
npm start
```
